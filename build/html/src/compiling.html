Compiling source code¶

There are two easy options:

  • Source -> Object -> Executable

  • Source -> Executable

Option

Command

Source -> Object -> Executable

gfortran -c hello.f90; gfortran exe -o hello.o

Source -> Executable

gfortran -o exe hello.f90

Dependencies¶

  • gfortran (or another Fortran compiler) is a required dependency
